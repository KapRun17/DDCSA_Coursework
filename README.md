# Платформа для поиска команд в онлайн-игры

Курсовая работа по дисциплине "Проектирование и разработка клиент-серверных приложений".

## Структура проекта

- `/server` - Серверная часть (Node.js + Express)
- `/client` - Клиентская часть (React)
- `/docs` - Документация и отчет

## Стек технологий

- MongoDB
- Express.js
- React.js
- Node.js
